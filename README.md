# "Adios mundo" en Java

## Descripción

Este programa imprime una vez el mensaje Adiós Mundo por pantalla

## Cómo compilar y ejecutar el programa

Para **compilarlo** escribimos desde la consola (necesitamos tener instalado *JDK*)

`$ javac AdiosMundo.java`

Para **ejecutarlo**

`$ java AdiosMundo`
